# Third-party notices

Status: Phase 0

AuroraSMS contains no production code or bundled third-party library at the
Phase 0 documentation gate.

The exact pre-implementation allowlist is in `docs/DEPENDENCY_POLICY.md`.
Entries are generated and reviewed when Phase 1 first resolves those artifacts.
Every distributed dependency entry will include
its name, exact version, project URL, copyright notice, license identifier,
license text location, purpose, and any AuroraSMS modifications.

Android SDK, AndroidX, Kotlin, Gradle, test tools, and build plugins are not
approved merely by appearing in a toolchain. Only the exact allowlisted
versions may resolve, and their notices are added from the resolved graph.

The private AuroraSMS handoff, private screenshots, and original source
artwork are not currently distributed repository content. Artwork ownership
and publication rights remain unresolved; artwork may enter the repository
only after its separate owner-approved license and attribution are recorded in
`docs/ARTWORK_CATALOG.md`.
